# Настройка иконок RapidWhisper

## Что было сделано

### 1. Генерация PNG иконок
- Создан скрипт `generate-icon.py` для генерации иконок с буквами "RW"
- Генерируются 3 размера: 16x16, 48x48, 128x128 пикселей
- Иконки сохраняются в `public/icons/`
- Цвет фона: Indigo (#4F46E5)
- Цвет текста: Белый
- Шрифт: Arial Bold (Windows)

### 2. Создание .ico файла
- Создан скрипт `create-ico.py` для конвертации PNG в .ico
- Файл `public/RapidWhisper.ico` содержит все размеры
- Используется для Windows (.exe и системный трей)

### 3. Интеграция в приложение
- **Системный трей**: `ui/tray_icon.py` загружает `RapidWhisper.ico`
- **PyInstaller**: `RapidWhisper.spec` включает иконку в сборку
- **Исполняемый файл**: .exe будет иметь иконку "RW"

## Использование

### Генерация иконок
```bash
# Установить Pillow (если еще не установлен)
uv add pillow

# Сгенерировать PNG иконки
uv run python generate-icon.py

# Создать .ico файл
uv run python create-ico.py
```

### Сборка с иконками
```bash
# Собрать .exe с иконками
build.bat
```

## Файлы

- `generate-icon.py` - Генератор PNG иконок
- `create-ico.py` - Конвертер PNG → ICO
- `public/icons/icon16.png` - Иконка 16x16
- `public/icons/icon48.png` - Иконка 48x48
- `public/icons/icon128.png` - Иконка 128x128
- `public/RapidWhisper.ico` - Windows иконка (все размеры)

## Где используются иконки

1. **Системный трей** - иконка "RW" в панели задач
2. **Исполняемый файл** - иконка .exe файла в проводнике
3. **Окна приложения** - иконка в заголовке окон (автоматически)

## Изменение дизайна иконки

Чтобы изменить дизайн иконки:

1. Отредактируйте `generate-icon.py`:
   - Измените цвет фона: `(79, 70, 229, 255)` → ваш RGBA цвет
   - Измените текст: `text = "RW"` → ваш текст
   - Измените цвет текста: `(255, 255, 255, 255)` → ваш RGBA цвет

2. Запустите генерацию:
   ```bash
   uv run python generate-icon.py
   uv run python create-ico.py
   ```

3. Пересоберите приложение:
   ```bash
   build.bat
   ```
