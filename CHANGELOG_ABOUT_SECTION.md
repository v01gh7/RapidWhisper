# Changelog: About Section Update

## Дата: 30 января 2026

### Изменения в разделе "О программе"

#### 1. Удалено упоминание SuperWhisper
- Убрана строка о вдохновении проектом SuperWhisper
- Теперь RapidWhisper представлен как самостоятельный проект

#### 2. Добавлен список используемых библиотек

**Основные библиотеки:**
- **PyQt6** - GUI фреймворк для создания интерфейса
- **OpenAI Python SDK** - Клиент для работы с AI API
- **PyAudio** - Запись аудио с микрофона
- **NumPy** - Обработка аудио данных
- **Keyboard** - Обработка глобальных горячих клавиш
- **Pyperclip** - Работа с буфером обмена
- **Psutil** - Управление процессами (single instance)
- **Python-dotenv** - Загрузка конфигурации из .env

**Библиотеки для тестирования:**
- **Pytest** - Фреймворк для unit-тестирования
- **Hypothesis** - Property-based testing

Все библиотеки в разделе "О программе" теперь имеют кликабельные ссылки на официальные сайты.

#### 3. Добавлен список поддерживаемых AI провайдеров

**Облачные провайдеры:**
- **Groq** - Бесплатный и быстрый API (рекомендуется)
- **OpenAI** - Официальный Whisper API
- **GLM (Zhipu AI)** - Поддержка китайского языка

**Локальные провайдеры (Custom):**
- **LM Studio** - Простой запуск локальных моделей
- **Ollama** - CLI для локальных моделей
- **vLLM** - Высокопроизводительный inference
- **LocalAI** - Локальная альтернатива OpenAI
- **Любые OpenAI-совместимые API**

#### 4. Настраиваемые ссылки через .env

Добавлены новые поля в конфигурацию:

```env
# ============================================
# About Section Links (OPTIONAL)
# ============================================
# GitHub repository URL
GITHUB_URL=https://github.com/yourusername/rapidwhisper

# Documentation URL
DOCS_URL=https://github.com/yourusername/rapidwhisper/tree/main/docs
```

**Как использовать:**
1. Откройте файл `.env` (или `%APPDATA%/RapidWhisper/.env` для production)
2. Найдите секцию "About Section Links"
3. Замените URL на свои:
   - `GITHUB_URL` - ссылка на ваш GitHub репозиторий
   - `DOCS_URL` - ссылка на документацию

**Пример:**
```env
GITHUB_URL=https://github.com/myusername/rapidwhisper
DOCS_URL=https://rapidwhisper.readthedocs.io
```

Ссылки автоматически обновятся в разделе "О программе" после перезапуска приложения или применения настроек.

### Для пользователей

После обновления:
1. Откройте настройки приложения
2. Перейдите в раздел "ℹ️ О программе"
3. Проверьте список библиотек и провайдеров
4. Все ссылки кликабельны и открываются в браузере

Чтобы изменить ссылки на GitHub и документацию:
1. Откройте `.env` файл
2. Найдите секцию "About Section Links"
3. Измените URL на свои
4. Перезапустите приложение

### Технические детали

- Все ссылки используют `setOpenExternalLinks(True)` для открытия в браузере
- URL загружаются при старте приложения из .env
- Если URL не указаны в .env, используются значения по умолчанию
- Поддержка hot reload - изменения применяются после сохранения настроек
