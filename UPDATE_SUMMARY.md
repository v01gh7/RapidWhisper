# Обновление: Раздел "О программе" и адаптивное окно настроек

## Дата: 30 января 2026

## Краткое описание

Обновлен раздел "О программе" в настройках приложения и добавлена поддержка адаптивной высоты окна с красивым скроллом.

## Что нового?

### 1. ✅ Убрано упоминание SuperWhisper
Приложение теперь представлено как самостоятельный проект без ссылок на вдохновение.

### 2. ✅ Добавлен полный список библиотек
**Основные:**
- PyQt6, OpenAI SDK, PyAudio, NumPy, Keyboard, Pyperclip, Psutil, Python-dotenv

**Тестирование:**
- Pytest, Hypothesis

Все библиотеки имеют кликабельные ссылки на официальные сайты.

### 3. ✅ Список поддерживаемых AI провайдеров
**Облачные:** Groq, OpenAI, GLM  
**Локальные (Custom):** LM Studio, Ollama, vLLM, LocalAI, любые OpenAI-совместимые

### 4. ✅ Настраиваемые ссылки через .env
Добавлены поля в `.env`:
```env
GITHUB_URL=https://github.com/yourusername/rapidwhisper
DOCS_URL=https://github.com/yourusername/rapidwhisper/tree/main/docs
```

Теперь вы можете указать свои ссылки на GitHub и документацию!

### 5. ✅ Адаптивная высота окна с красивым скроллом
- Максимальная высота = высота экрана - 160px
- Красивый скролл в стиле macOS
- Автоматическое появление при необходимости
- Работает на всех страницах настроек

## Как использовать новые возможности?

### Изменить ссылки на GitHub и документацию:
1. Откройте `.env` файл (или `%APPDATA%/RapidWhisper/.env`)
2. Найдите секцию "About Section Links"
3. Замените URL на свои
4. Перезапустите приложение

### Просмотреть информацию о программе:
1. Откройте настройки (через трей или горячую клавишу)
2. Выберите "ℹ️ О программе"
3. Прокрутите страницу, чтобы увидеть все библиотеки и провайдеры
4. Кликните на любую ссылку, чтобы открыть в браузере
