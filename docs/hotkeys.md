# Горячие клавиши RapidWhisper

## Глобальные горячие клавиши

Эти клавиши работают из любого приложения, даже когда RapidWhisper свернут в трей.

### Основные

| Клавиша | Действие | Описание |
|---------|----------|----------|
| **Ctrl+Space** | Начать/Остановить запись | Первое нажатие начинает запись, второе - останавливает и отправляет на транскрипцию |
| **ESC** | Отменить запись | Отменяет текущую запись без транскрипции (работает только во время записи) |

## Меню трея

Правый клик на иконке в трее открывает меню:

| Пункт меню | Действие |
|------------|----------|
| **Настройки** | Открывает окно настроек |
| **Выход** | Закрывает приложение |

## Настройка горячих клавиш

Вы можете изменить горячую клавишу в окне настроек или в файле `.env`:

### Примеры комбинаций

```env
# Одна клавиша
HOTKEY=F1
HOTKEY=F2

# Ctrl + клавиша
HOTKEY=ctrl+space
HOTKEY=ctrl+r
HOTKEY=ctrl+shift+r

# Alt + клавиша
HOTKEY=alt+space
HOTKEY=alt+v

# Shift + клавиша
HOTKEY=shift+space
```

### Поддерживаемые клавиши

- **Функциональные**: F1-F12
- **Буквы**: a-z
- **Цифры**: 0-9
- **Модификаторы**: ctrl, alt, shift
- **Специальные**: space, enter, tab, esc

### Рекомендации

- **Для быстрого доступа**: `ctrl+space`, `alt+space`
- **Для избежания конфликтов**: `F1`, `F2`, `F3`
- **Для удобства**: `ctrl+shift+r`, `alt+v`

⚠️ **Внимание**: Убедитесь что выбранная комбинация не используется другими приложениями!

## Автоматические действия

Некоторые действия происходят автоматически без нажатия клавиш:

| Событие | Действие |
|---------|----------|
| **Тишина обнаружена** | Автоматически останавливает запись после настроенной длительности тишины |
| **Транскрипция завершена** | Автоматически копирует текст в буфер обмена |
| **Окно показано** | Автоматически скрывается через настроенную задержку (если не используется) |

## Советы по использованию

### Быстрый рабочий процесс

1. **Ctrl+Space** - начать запись
2. Говорите
3. **Ctrl+Space** - остановить и отправить
4. **Ctrl+V** - вставить результат

### Отмена ошибочной записи

1. **Ctrl+Space** - начать запись
2. Поняли что ошиблись
3. **ESC** - отменить без транскрипции

### Работа с длинными записями

1. **Ctrl+Space** - начать запись
2. Говорите
3. Сделайте паузу 1.5+ секунды
4. Запись автоматически остановится

## Конфликты горячих клавиш

Если горячая клавиша не работает, возможно она используется другим приложением:

### Windows

Часто используемые комбинации:
- `Win+Space` - переключение языка
- `Ctrl+Shift+Esc` - диспетчер задач
- `Alt+Tab` - переключение окон

### Рекомендуемые безопасные комбинации:
- `F1`, `F2`, `F3` (если не используются в текущем приложении)
- `Ctrl+Space` (обычно свободна)
- `Alt+Space` (может конфликтовать с системным меню)

### Проверка конфликтов

1. Откройте настройки
2. Попробуйте другую комбинацию
3. Сохраните и перезапустите приложение
4. Проверьте работу новой комбинации
