RapidWhisper
Для чего Хоткеи Фичи Провайдеры Пайплайн

Global Speech-to-Text Desktop Tool

Говоришь в любом окне. Получаешь готовый текст сразу.

RapidWhisper нужен для быстрого голосового ввода без переключения контекста: запуск записи по хоткею, форматирование под активное приложение и готовый текст в буфере обмена.

  • Работает поверх любых окон и не крадет фокус.
  • Поддерживает встроенные и кастомные OpenAI-совместимые API.
  • Темы интерфейса переключаются мгновенно прямо на странице.

Для чего программа

Диктовка без переключения окон

Запустил запись, проговорил, вставил текст там же, где работал.

Единый формат вывода

Фиксируешь формат и получаешь одинаковый стиль текста в любом приложении.

Гибкая автоматизация

Hooks позволяют добавлять свои действия до и после обработки текста.

Пайплайн

  1. 01 Hotkey открывает floating-window и запускает запись.
  2. 02 Волна и статус обновляются в реальном времени.
  3. 03 Остановка по хоткею или тишине.
  4. 04 Форматирование под выбранный/определенный контекст.
  5. 05 Результат в буфере обмена + пост-обработка hooks.
