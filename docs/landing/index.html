Global Speech-to-Text Desktop Tool

Говоришь в любой программе. Получаешь готовый текст сразу в буфер.

RapidWhisper нужен для быстрой голосовой работы без прыжков между окнами: запись по хоткею, форматирование под контекст приложения, копирование результата и продолжение работы в том же месте.

Для чего программа

Диктовка без контекстного переключения

Нажимаешь хоткей в любом окне, говоришь, получаешь текст и вставляешь его там же. Без отдельного редактора и без ручной копипасты.

Контролируемое форматирование

Можно зафиксировать целевой формат заранее или запустить ручное форматирование для любого текста по отдельному хоткею.

Автоматизация вокруг текста

Hooks-пайплайн позволяет встраивать свои шаги до/после записи и форматирования: проверка, пост-обработка, отправка в внешние инструменты.

Пайплайн работы

  1. 01 Hotkey открывает floating-window и начинает запись.
  2. 02 В реальном времени идёт визуализация, отслеживается активное окно.
  3. 03 Остановка по хоткею или по тишине (если включено).
  4. 04 Текст проходит форматирование под выбранный/определённый контекст.
  5. 05 Результат копируется в буфер и может быть обработан hooks-скриптами.

before_recording, after_recording, transcription_received и другие этапы пайплайна.
