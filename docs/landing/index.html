Горячие клавиши и ручное форматирование

Управляйте форматом напрямую. Можно закрепить один формат для всех транскрипций или вручную прогнать любой текст через выбранный стиль.

Запись

Ctrl Space

Старт и стоп записи с отправкой на транскрипцию.

Отмена

Esc

Мгновенно отменяет запись без обработки текста.

Выбор формата

Ctrl Alt Space

Выбираешь формат - и дальше он применяется всегда, независимо от приложения.

Пример: сидишь в Notion, но выбрал BBCode - получишь BBCode.

Ручное форматирование

Ctrl Shift Space

Открывает окно, куда можно вставить любой текст.

Нажимаешь - и получаешь этот текст в выбранном формате.

Global Hotkey

Ctrl+Space запускает и останавливает запись из любого приложения.

Format Picker

Фиксирует формат: все транскрипции идут в выбранном стиле.

Manual Format

Любой текст можно быстро прогнать через нужный формат без записи.

Cancel Safe

Esc мгновенно отменяет запись без транскрипции.

Процесс без лишних шагов

Запустите запись, говорите, остановите по Ctrl+Space или дождитесь авто-детекта тишины. RapidWhisper сам выберет формат и скопирует результат.

01

Запуск

Горячая клавиша открывает плавающее окно в любом приложении.

02

Фиксация

Инфопанель держит контекст и подсказки управления.

03

Отправка

Текст формируется под приложение и копируется без окон.
