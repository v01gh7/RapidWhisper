Introduction

RapidWhisper is a modern speech-to-text transcription application that turns your voice into perfect text instantly.

Cross-Platform Support

RapidWhisper works seamlessly on Windows, macOS, and Linux with native features for each platform.

Key Features

  • Lightning Fast: 1-2 second transcription with Groq, OpenAI, or GLM APIs
  • Smart Formatting: Automatically formats text based on active application
  • Multi-Language: Interface in 15 languages, speak in any language
  • Beautiful UI: Modern floating window with real-time waveform
  • Global Hotkey: Activate from anywhere with customizable shortcuts

Installation

Windows

  1. Download RapidWhisper.exe from the downloads page
  2. Run the executable - no installation required
  3. Follow the welcome screen instructions
  4. Settings are saved in %APPDATA%\RapidWhisper\.env

macOS

  1. Download RapidWhisper.dmg
  2. Open the DMG and drag RapidWhisper to Applications
  3. Grant accessibility permissions when prompted
  4. Settings are saved in ~/Library/Application Support/RapidWhisper/.env

Linux

  1. Download RapidWhisper.AppImage
  2. Make it executable: chmod +x RapidWhisper.AppImage
  3. Run: ./RapidWhisper.AppImage
  4. Settings are saved in ~/.config/RapidWhisper/.env

Global Hotkey

Activate RapidWhisper from anywhere with a customizable keyboard shortcut.

Default Hotkey

The default hotkey is Ctrl+Space (or Cmd+Space on macOS).

Changing the Hotkey

  1. Right-click the system tray icon
  2. Select "Settings"
  3. Go to the "Application" tab
  4. Click the hotkey field
  5. Press your desired key combination
  6. Click "Save"

Supported Modifiers

  • Ctrl - Control key
  • Alt - Alt key
  • Shift - Shift key
  • Cmd - Command key (macOS only)
‚ö†Ô∏è
Hotkey Conflicts

If your hotkey doesn't work, another application may be using it. Try a different combination like F2 or Ctrl+Alt+Space.

Troubleshooting

Microphone Issues

Error: "Microphone is busy"

  • Close other applications using the microphone
  • Check system audio settings
  • Restart RapidWhisper

API Issues

Error: "Invalid API key"

  • Verify your API key is correct
  • Check for extra spaces
  • Ensure the key is for the correct provider

Error: "Network error"

  • Check your internet connection
  • Verify the API service is accessible
  • Check firewall settings

Hotkey Issues

Hotkey not working

  • Try a different key combination
  • Check if another app is using the same hotkey
  • Run as administrator (Windows)
  • Grant accessibility permissions (macOS)

Frequently Asked Questions

Which AI provider should I use?

Groq is recommended for beginners - it's free and fast. OpenAI offers the best quality. GLM is great for Chinese language support.

How do I change the hotkey?

Right-click the tray icon ‚Üí Settings ‚Üí Application tab ‚Üí Change hotkey ‚Üí Save.

Why does recording stop too quickly?

Increase "Silence Duration" in Settings ‚Üí Application tab. Try 2.0-2.5 seconds.

Can I use RapidWhisper offline?

No, RapidWhisper requires an internet connection to use AI transcription APIs.

Where are my settings stored?

Windows: %APPDATA%\RapidWhisper\.env
macOS: ~/Library/Application Support/RapidWhisper/.env
Linux: ~/.config/RapidWhisper/.env

Is my voice data stored?

No, audio is sent directly to the AI provider for transcription and not stored by RapidWhisper.
