# Руководство: Дизайн заголовков групп

## Что изменилось?

Заголовки групп в настройках теперь имеют улучшенный дизайн - они хорошо видны, современные и привлекают внимание.

## Визуальное сравнение

### ❌ Старый дизайн (плохо)

```
┌────────────────────────────────────────┐
│ провайдер                              │ ← Серый, мелкий, без фона
│ ┌────────────────────────────────────┐ │
│ │ Провайдер: [groq ▼]               │ │
│ └────────────────────────────────────┘ │
└────────────────────────────────────────┘
```

**Проблемы:**
- Серый цвет (#888888) - плохо видно
- Мелкий шрифт (11px)
- Нет фона - сливается с окружением
- Все заглавные буквы - трудно читать
- Нет акцента

### ✅ Новый дизайн (хорошо)

```
┌────────────────────────────────────────┐
│ ┏━━━━━━━━━━━━━━━┓                      │
│ ┃ Провайдер     ┃                      │ ← Белый, крупный, с фоном
│ ┗━━━━━━━━━━━━━━━┛                      │
│ ┌────────────────────────────────────┐ │
│ │ Провайдер: [groq ▼]               │ │
│ └────────────────────────────────────┘ │
└────────────────────────────────────────┘
```

**Преимущества:**
- Белый цвет (#ffffff) - отлично видно
- Крупный шрифт (13px)
- Темно-серый фон (#2d2d2d) - выделяется
- Нормальный регистр - легко читать
- Синяя рамка (#0078d4) - акцент

## Технические детали

### Стили CSS

```css
QGroupBox::title {
    /* Позиционирование */
    subcontrol-origin: margin;
    subcontrol-position: top left;
    
    /* Отступы */
    padding: 8px 16px;
    
    /* Фон и рамка */
    background-color: #2d2d2d;
    border: 1px solid #0078d4;
    border-radius: 4px;
    
    /* Текст */
    color: #ffffff;
    font-size: 13px;
    font-weight: bold;
}
```

### Цветовая палитра

| Элемент | Цвет | Hex | Описание |
|---------|------|-----|----------|
| Текст | Белый | `#ffffff` | Основной цвет заголовка |
| Фон | Темно-серый | `#2d2d2d` | Фон заголовка |
| Рамка | Синий | `#0078d4` | Акцентная рамка |
| Фон группы | Очень темный | `#252525` | Фон всей группы |

### Размеры

| Параметр | Значение | Описание |
|----------|----------|----------|
| Размер шрифта | 13px | Крупнее обычного текста (12px) |
| Отступы | 8px 16px | Вертикальные и горизонтальные |
| Скругление | 4px | Радиус углов |
| Рамка | 1px | Толщина синей рамки |
| Отступ сверху | 20px | Между группами |

## Контрастность и доступность

### Тест контрастности

**Заголовок на фоне:**
- Текст: `#ffffff` (белый)
- Фон: `#2d2d2d` (темно-серый)
- **Контраст: 12:1** ✅

**Требования WCAG 2.1:**
- Уровень AA: 4.5:1 (обычный текст)
- Уровень AAA: 7:1 (обычный текст)
- **Наш результат: 12:1** (превышает AAA)

### Читаемость

✅ **Отлично читается** при:
- Нормальном освещении
- Ярком свете
- Слабом освещении
- Для людей с нарушениями зрения

- **После**: Крупный текст, нормальный регистр

## Тестирование

### Как проверить

1. Откройте настройки (трей → Настройки)
2. Перейдите на любую страницу
3. Посмотрите на заголовки групп

### Что проверить

✅ Заголовки белые (не серые)  
✅ Заголовки крупные (13px)  
✅ Заголовки жирные (bold)  
✅ Заголовки имеют фон (#2d2d2d)  
✅ Заголовки имеют синюю рамку  
✅ Заголовки имеют скругленные углы  
✅ Заголовки хорошо видны
