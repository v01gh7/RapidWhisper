# Руководство: Новое поле ввода горячей клавиши

## Что изменилось?

Теперь в настройках приложения поле для выбора горячей клавиши работает по-новому!

## Как использовать

### Старый способ (больше не работает)
❌ Вручную вводить текст: `ctrl+space`

### Новый способ (правильный)
✅ Просто нажать нужное сочетание клавиш!

## Пошаговая инструкция

1. **Откройте настройки** (иконка в трее → Настройки)
2. **Перейдите в раздел "Приложение"**
3. **Кликните на поле "Горячая клавиша"**
4. **Нажмите нужное сочетание клавиш** (например: Ctrl+Space)
5. **Сочетание автоматически запишется** в поле
6. **Нажмите "Сохранить"**

## Примеры

### Простые клавиши
- Нажмите `F1` → в поле появится: `f1`
- Нажмите `F12` → в поле появится: `f12`

### С модификаторами
- Нажмите `Ctrl+Space` → в поле появится: `ctrl+space`
- Нажмите `Ctrl+Shift+R` → в поле появится: `ctrl+shift+r`
- Нажмите `Alt+F4` → в поле появится: `alt+f4`

### Специальные клавиши
- Нажмите `Ctrl+Enter` → в поле появится: `ctrl+enter`
- Нажмите `Shift+Tab` → в поле появится: `shift+tab`

## Поддерживаемые клавиши

### ✅ Работают
- **Модификаторы**: Ctrl, Shift, Alt
- **Функциональные**: F1-F12
- **Специальные**: Space, Enter, Tab, Esc, Delete, Insert, Home, End, Page Up, Page Down
- **Стрелки**: Up, Down, Left, Right
- **Буквы**: A-Z
- **Цифры**: 0-9

### ❌ Не работают
- Только модификаторы без основной клавиши (например, просто Ctrl)
- Экзотические клавиши (Pause, Print Screen, и т.д.)

## Важно!

- **Нельзя вводить текст вручную** - поле только для чтения
- **Нужно именно нажать клавиши** - не печатать их название
- **Сочетание записывается сразу** - не нужно нажимать Enter

## Если что-то не работает

1. Убедитесь, что вы **кликнули на поле** (оно должно быть в фокусе)
2. **Нажмите нужное сочетание** (не печатайте текст)
3. Если сочетание не записалось, попробуйте другое

## Преимущества

✅ **Быстрее** - не нужно вводить текст вручную  
✅ **Точнее** - исключены опечатки  
✅ **Удобнее** - просто нажми и готово  
✅ **Интуитивнее** - работает как ожидается  

## Тестирование

Чтобы протестировать виджет отдельно, запустите:
```bash
uv run python test_hotkey_input.py
```

Это откроет тестовое окно, где можно попробовать разные сочетания клавиш.
