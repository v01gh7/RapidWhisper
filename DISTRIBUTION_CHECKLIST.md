# Checklist для распространения RapidWhisper

## Перед сборкой

- [ ] Все тесты проходят (`uv run pytest`)
- [ ] Версия обновлена в коде
- [ ] CHANGELOG.md обновлен
- [ ] Документация актуальна
- [ ] .env.example содержит все параметры
- [ ] .env.example НЕ содержит реальных ключей

## Сборка

- [ ] Запущен `build.bat`
- [ ] Сборка завершилась без ошибок
- [ ] .exe файл создан в `dist/`
- [ ] Ваш .env восстановлен из backup
- [ ] Размер .exe разумный (~150-200 MB)

## Проверка безопасности

- [ ] ✅ **КРИТИЧНО**: .exe НЕ содержит ваших API ключей
- [ ] Проверено: открыт .exe в архиваторе, нет .env с ключами
- [ ] Проверено: только .env.example с пустыми полями
- [ ] Логи не содержат чувствительной информации

## Тестирование .exe

- [ ] Скопирован .exe в чистую папку
- [ ] Создан .env с тестовым ключом
- [ ] .exe запускается без ошибок
- [ ] Иконка появляется в трее
- [ ] Окно настроек открывается
- [ ] Горячие клавиши работают
- [ ] Запись аудио работает
- [ ] Транскрипция работает
- [ ] Результат копируется в буфер
- [ ] Уведомления в трее работают
- [ ] Приложение закрывается корректно

## Подготовка пакета для распространения

Создайте папку `RapidWhisper-v1.1.0/`:

- [ ] `RapidWhisper.exe` - исполняемый файл
- [ ] `.env.example` - пример настроек
- [ ] `INSTALL.txt` - инструкция по установке
- [ ] `README.txt` - краткое описание (опционально)

## Создание архива

- [ ] Создан ZIP: `RapidWhisper-v1.1.0.zip`
- [ ] Размер архива разумный (~50-80 MB)
- [ ] Архив открывается без ошибок
- [ ] Все файлы на месте

## Финальная проверка

- [ ] Распакован архив в новую папку
- [ ] Создан .env с тестовым ключом
- [ ] Запущен .exe
- [ ] Все функции работают
- [ ] Нет ошибок в логах

## Документация для пользователей

- [ ] INSTALL.txt понятен и актуален
- [ ] Ссылки на получение API ключей работают
- [ ] Инструкции по решению проблем актуальны
- [ ] Контактная информация указана (если нужно)

## Распространение

- [ ] Загружен на GitHub Releases (если используется)
- [ ] Создан changelog для релиза
- [ ] Указана версия и дата
- [ ] Добавлены скриншоты (опционально)

## После релиза

- [ ] Протестирован на чистой системе
- [ ] Получена обратная связь от пользователей
- [ ] Исправлены критические баги (если есть)
- [ ] Обновлена документация на основе вопросов
